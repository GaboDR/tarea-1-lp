Tarea 1 de Lenguajes de Programación
=============================================
--------------------------------------------------------------------
IMPORTANTE SOBRE PAQUETES USADOS
--------------------------------------------------------------------
- Los paquetes usados en el código no necesitaron instalación
    previa.

--------------------------------------------------------------------
AMBIENTE DE EJECUCION USADO
--------------------------------------------------------------------
- Sistema operativo: Windows 11
- Ejecución del código: Visual Studio Code

--------------------------------------------------------------------
EJECUCIÓN
--------------------------------------------------------------------
- La ejecución se hizo por la terminal de Visual Studio Code a 
    través del Run Python File del editor de texto

--------------------------------------------------------------------
CONSIDERACIONES AL MOMENTO DE EJECUTAR
--------------------------------------------------------------------
- El archivo funciones.py contiene las funciones para poder ejecutar
    correctamente el interprete, debe estar en la misma carpeta que
    interprete.py 

- Se debe tener el archivo .txt en la misma carpeta que el código a 
    ejecutar para que pueda leerlo
    
- El archivo que contiene el código a interpretar se debe de llamar
    "codigo.txt" para que sea ejecutado correctamente

- Si usa la función mostrar, generará un archivo llamado "output.txt"
    que contendrá el valor de la variable que se le entregue como
    parámetro
